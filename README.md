# 🍔 StackBurgers - Mucho más que una Landing Page

¡Hola! Bienvenido al repositorio de **StackBurgers**. 

Si estás leyendo esto, probablemente estés evaluando mi proyecto para el bootcamp de **Generation Chile**, o quizás simplemente llegaste aquí navegando por GitHub. De cualquier forma, quiero darte un poco de contexto sobre lo que estás a punto de ver.

## 📖 La historia detrás del código

Siempre he creído que programar tiene muchísimo de cocinar: tomas ingredientes crudos (HTML, CSS, JS), pruebas distintas recetas, a veces las cosas se "queman" en el proceso y te toca empezar de nuevo, pero la satisfacción de ver el resultado final funcionando en pantalla no tiene precio.

Este proyecto marca un hito importante para mí. Representa el cierre del módulo de fundamentos web, donde pasamos de entender etiquetas sueltas en una consola a construir algo real, visual y funcional. Hubo momentos de frustración peleando con elementos que no se centraban o lógica que no quería ejecutar, pero ver cómo la página iba cobrando vida ha sido un proceso increíblemente gratificante.

## ✨ El ingrediente secreto: Yendo más allá del DOM

La rúbrica del desafío pedía cumplir con bases sólidas (manipulación del DOM, eventos, diseño con Bootstrap). Sin embargo, tras conversarlo en clase y tener luz verde para explorar nuevas herramientas, decidí ponerme un reto personal mayor. No quería hacer "solo otra plantilla".

Quería darle a la página esa sensación *premium* y fluida que solemos ver en sitios web de alta gama. Para lograrlo, decidí salir un poco de mi zona de confort e integrar **GSAP (GreenSock)** junto con su plugin **ScrollTrigger**. 

El resultado es un efecto donde los ingredientes principales reaccionan de manera fluida al scroll del usuario. Costó horas extra de lectura de documentación, prueba y error, pero siento que le dio al proyecto un alma completamente distinta y me demostró lo potente que puede ser la web moderna.

## 🎬 Inspiración y Referencia Visual

El efecto de scroll implementado en la sección Hero está inspirado en animaciones de productos de alto nivel de la industria (estilo las secuencias de scroll de Apple). 

El objetivo principal fue replicar el ensamblaje visual de la hamburguesa a medida que el usuario interactúa con la página, pasando de una maqueta estática a una experiencia inmersiva. 

## 🛠️ ¿Con qué "cocinamos" este proyecto? (Stack Tecnológico)

* **Estructura:** HTML5 Semántico.
* **Diseño Responsive:** CSS3 + **Bootstrap 5.3**. Aproveché su sistema de grillas, navbar interactivo y componentes para asegurar un diseño *mobile-first* impecable.
* **Interactividad Core:** Vanilla JavaScript (ES6+). Aquí desarrollé la lógica de reservas, captura de datos del formulario, modificación de estilos en tiempo real y eventos nativos.
* **La Magia Visual:** GSAP & ScrollTrigger para las animaciones avanzadas.

## 🎯 Requisitos del desafío cumplidos a cabalidad:
- [x] Diseño responsivo (adaptable a móviles y escritorio).
- [x] Navbar funcional con menú hamburguesa integrado.
- [x] Sección Hero inmersiva con imagen de fondo y call-to-action.
- [x] Sección de Menú utilizando *Cards* bien distribuidas en columnas.
- [x] Formulario de reservas interactivo, limpio y estilizado.
- [x] JavaScript nativo: uso de `onclick`, captura de `value` del formulario, `alert()` dinámico y modificación de propiedades desde JS.
- [x] Integración de buenas prácticas de arquitectura de archivos y control de versiones.
